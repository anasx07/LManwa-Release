# LManwa-Release

The official distribution repository for LManwa.

## 🚀 Experience the Premium
LManwa provides a high-quality experience directly on your Windows device. 

## 📥 Download
You can always find the latest version on our [Releases page](https://github.com/anasx07/LManwa-Release/releases/latest).

## 🛠️ Development
This landing page is built with React + Vite.

### Build & Deploy
If you are making changes to the landing page:
1. `npm install`
2. `npm run build`
3. `npm run deploy` (This builds and pushes the `dist` folder to the `gh-pages` branch)
